Memoria RAM

La memoria RAM (Random Access Memory) es un componente esencial en cualquier computadora. Es responsable de almacenar temporalmente los datos que el procesador necesita para ejecutar aplicaciones y tareas.

Características principales

  • Capacidad: La cantidad de datos que puede almacenar, medida en GB (gigabytes).
  • Velocidad: La velocidad de transferencia de datos, medida en MHz (megahercios).
  • Tipo: DDR4, DDR5, etc., que definen la generación y eficiencia de la RAM.
  • Latencia: El tiempo que tarda en responder a las solicitudes del procesador.

Tipos de Memoria RAM

Existen varios tipos de memoria RAM, cada uno con sus propias características y usos:

  • DDR3
  • DDR4
  • DDR5
  • RAM ECC (Error-Correcting Code)
  • SO-DIMM (para laptops)

Marcas destacadas de Memoria RAM

  • Corsair
  • Kingston
  • G.Skill
  • Crucial
  • HyperX
