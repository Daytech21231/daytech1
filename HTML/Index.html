COMPONENTES ELECTRONICOS

En tu vida diaria has usado varios sistemas tecnológicos como pantallas, consolas, celulares y computadoras.

¿Pero alguna vez te has preguntado cómo está creado el sistema tecnológico que estás usando?

Todos estos funcionan a partir de componentes que están conectados o colocados a lo que conocemos como Motherboard.


Todos estos componentes hacen que tu dispositivo funcione, ya que el Software manda las instrucciones de qué debe hacer cada componente para que tu sistema operativo arranque.

Los componentes más comunes son:

  • Procesador (CPU)
  • Tarjeta gráfica (GPU)
  • Memoria RAM
  • Disco duro (HDD) o Unidad de estado sólido (SSD)
  • Placa base (Motherboard)
  • Fuente de alimentación (PSU)
